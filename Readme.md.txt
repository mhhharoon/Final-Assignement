# HTML & CSS Projects - Haroon Flooring

Welcome to my GitHub repository for HTML and CSS projects! This repository contains all the projects that I have completed during the course. The goal of these projects was to enhance my understanding of web development, focusing on HTML structure and CSS styling.

---

## Projects Completed

### 1. **Haroon Flooring Website**
A multi-page website for a fictional company called *Haroon Flooring*. This project showcases various aspects of HTML and CSS design, including creating a basic layout, adding a navigation bar, setting background images, and structuring content.

#### Features:
- **Home Page**: A welcoming section with a background image and a brief description of the company.
- **Services Page**: A page that describes the various flooring services offered.
- **About Us Page**: Provides background information about the company and its experience in the industry.
- **Contact Page**: A contact form and contact information, including email and phone links.
- **Responsive Navigation**: A navigation bar with links to the Home, Services, About, and Contact pages.
- **Background Image**: Added a background image to the homepage for better user experience and aesthetic appeal.
  
You can view the [Haroon Flooring Website](https://github.com/haroonflooring/website).

#### Technologies Used:
- **HTML5**: For creating the structure of the website.
- **CSS3**: For styling the webpage, including layout, fonts, colors, and positioning.
- **Responsive Design**: Ensures the site is mobile-friendly and adjusts to different screen sizes.
